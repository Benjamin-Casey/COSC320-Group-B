## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Benjamin-Casey/COSC320-Group-B.git
   ```
2. Run docker container
   ```sh
   docker compose up -d
   ```
